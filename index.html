Featured technologies

KNX

KNX is the worldwide STANDARD for all applications in home automation and building control

EnOcean

EnOcean is an energy harvesting wireless technology used primarily in building automation systems

Insteon

INSTEON is a powerline based networking technology for the connected home..

HomeMatic

HomeMatic is a wireless home automation system developed and designed by eQ-3 AG

digitalSTROM

digitalSTROM represents digital lifestyle and integrated connectivity concept for intelligent living

MAX! Cube

The computer network interface for the radio components of the MAX! heating control

Koubachi

Koubachi analyzes your plant data and gives you instructions when and how to care for your plants

Asterisk

Asterisk is a free and open source framework for building communications applications

Tinkerforge

Tinkerforge is a platform of microcontroller building blocks that can control different modules

MQTT

MQTT is an open message protocol for M2M that enables the transfer of telemetry-style data

OpenSprinkler

OpenSprinkler is an open-source, Internet-based sprinkler controller built around the Arduino software

Xively

Xively is an on-line database service allowing developers to connect sensor-derived data to the Web
